Package rm::demographic

This document describes the architecture of the openEHR Demographic Information Model. The semantics are drawn from previous work in GEHR, existing models in CEN 13606 and the HL7v3 RIM, and other work done in Australia. The intended audience includes: • Standards bodies producing health informatics standards; • Software development groups using openEHR; • Academic groups using openEHR; • The open source healthcare community; • Medical informaticians and clinicians intersted in health information; • Health data managers.
