Abstract class defining the concept of true quantified values, i.e. values which are not only ordered, but which have a precise magnitude.

Attributes
SignatureOptionalityMultiplicityConstantDocumentation
 magnitude_status : String 0..1  --  Optional status of magnitude with values: • “=” : magnitude is a point value • “<“ : value is < magnitude • “>” : value is > magnitude • “<=” : value is <= magnitude • “>=” : value is >= magnitude • “~” : value is approximately magnitude If not present, meaning is “=”.
