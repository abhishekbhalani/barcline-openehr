Class DV_ABSOLUTE_QUANTITY{Abstract}


DV_QUANTIFIED

Direct Subclassifiers:

Abstract class defining the concept of quantified entities whose values are absolute with respect to an origin. Dates and Times are the main example.

Attributes
SignatureOptionalityMultiplicityConstantDocumentation
 accuracy : DV_AMOUNT 1  --  Accuracy of measurement, expressed as a half-range value of the diff type for this absolute quantity (i.e. an accuracy of x means +/−x).
